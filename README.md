# Window Bid Calculator

A mobile-first window cleaning quote calculator built for field use.

**Live URL:** https://YOUR-USERNAME.github.io/window-bid/

## Setup for Development

```bash
# Clone the repo, then serve locally:
npx serve .
```

Then open http://localhost:5000

## Deploy to GitHub Pages

1. Create a new repo on GitHub (public)
2. Push this code to the main branch
3. Go to Settings → Pages → Source: "main branch"
4. Your site will be live at `https://YOUR-USERNAME.github.io/REPO-NAME/`

## Features

- 11 window types with your custom pricing
- 3 service tiers: Basic, Premium, Luxury
- Per-window type increment/decrement counters
- Travel fee toggle (10% surcharge)
- Customer info capture
- Save bids locally
- Professional quote preview
- Mobile-optimized — install to home screen
